Hexo博客的搭建和部署

前置部署

下载Node.js

官网下载: Node.js (nodejs.org)

主要是要用它自带的npm工具, 具体可以看Vue脚手架搭建中的介绍

下载git

淘宝的git镜像: git-for-windows Mirror (taobao.org)

下载hexo-cli和本地运行

配置npm的镜像为淘宝的镜像

下载cnpm

下载hexo的脚手架

然后可以通过hexo s就可以再本地的localhost:4000端口测试代码

创建目录

创建一个hexo-blog文件夹, 进入cmd, 配置hexo-cli

启动本地的hexo服务

将hexo布置到github上

下载butterfly主题

在项目目录控制台输入下列代码下载主题
git clone -b master https://github.com/jerryc127/hexo-theme-butterfly.git themes/butterfly

安装pug 以及 stylus 的渲染器
npm install hexo-renderer-pug hexo-renderer-stylus --save

修改 Hexo 根目录下的 _config.yml，把主题改为butterfly

在hexo根目录下添加一个与_config平级的_config.butterfly.yml文件

文件内容复制themes/butterfly/_config.yml的内容

Hexo会自动合并主题中的_config.yml和 _config.butterfly.yml里的配置，如果存在同名配置，会使用_config.butterfly.yml的配置，其优先度较高
